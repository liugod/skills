# skills/content-output-SKILL.md

# 技能名称
Content Output Writer

# 目标
把已明确的内容任务写成本地可审核、可编辑、可发布的文件。

# 输入
- 上游 brief
- SOUL.md
- MEMORY.md
- sync/products.md
- sync/asset_index.md

# 输出原则
- 先输出完整内容包，而不是一句散文案
- 默认输出 markdown 文件
- 一次输出中应包含：
  - 标题候选
  - 核心文案
  - 配图/视频建议
  - CTA 建议
  - 风险提示（如有）

# 支持的输出类型
1. 社媒图文草稿
2. 短视频脚本
3. 口播稿
4. 海报文案
5. 发布会预热文案
6. 社区互动帖
7. 复盘总结

# 存储规则
根据内容类型写入：
- outputs/drafts/
- outputs/scripts/
- outputs/posters/
- outputs/review/

# 文件命名规则
YYYY-MM-DD_平台_主题_类型.md

# 特别规则
若任务是“小米式内容”，应默认满足：
- 至少有一个“用户价值”表达
- 至少有一个“场景化”表达
- 若适用，至少有一个“生态协同”表达
- 若为社区/活动内容，至少有一个“参与感”表达
