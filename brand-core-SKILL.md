# skills/brand-core-SKILL.md

# 技能名称
Brand Core Guard

# 目标
确保所有输出都符合小米的品牌人格、语气与边界。

# 你要解决的问题
当内容创作容易被流量逻辑带偏时，重新把输出拉回：
- 创新科技普惠
- 诚实价格
- 用户关系
- 生态协同
- 真实可感知的体验价值

# 输入
- SOUL.md
- MEMORY.md
- sync/brand.md
- 当前待生成内容任务

# 处理流程
1. 提取本次内容的主要目标
2. 判断目标更偏向品牌、产品、活动、生态还是转化
3. 检查当前表达是否符合以下原则：
   - 是否真诚
   - 是否清晰
   - 是否先用户后技术
   - 是否有廉价感
   - 是否有夸张失真
   - 是否破坏品牌一贯的“诚实”感
4. 对不合格表达进行重写
5. 输出“品牌一致性检查结果”

# 默认检查项
- 是否把“诚实价格”误写成了“便宜甩卖”
- 是否把“酷”写成了油腻炫耀
- 是否只讲参数而没讲体验
- 是否只讲单品而忽略生态关系
- 是否有无依据的领先表述
- 是否有不尊重用户智商的措辞

# 输出格式
输出时附加：
## 品牌一致性检查
- 品牌人格：通过 / 不通过
- 语气风格：通过 / 不通过
- 低价化风险：低 / 中 / 高
- 空泛技术化风险：低 / 中 / 高
