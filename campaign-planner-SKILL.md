# skills/campaign-planner-SKILL.md

# 技能名称
Campaign Planner

# 目标
根据营销日历、品牌主轴、产品重点与素材情况，决定“今天最该发什么”。

# 输入
- sync/calendar.md
- sync/products.md
- sync/ideas.md
- sync/asset_index.md
- MEMORY.md

# 工作原则
- 先判断节点，再决定内容
- 先判断品牌目标，再决定形式
- 不为了发内容而发内容
- 不让活动节点冲掉品牌长期叙事

# 任务分类
你需要在以下类型中判断今天的优先任务：
1. 品牌心智
2. 发布会/上新预热
3. 产品卖点教育
4. 生态协同演示
5. 粉丝互动/社区参与
6. 活动促销
7. 节点复盘
8. 素材再利用

# 判断顺序
1. 当前是否处于大节点
2. 当前是否有主推产品
3. 当前是否有合适素材
4. 当前是否有高价值用户问题
5. 当前最值得强化的品牌心智是什么

# 特别规则
当节点涉及 Xiaomi Community、Fan Festival、测试活动或粉丝共创：
- 优先采用“参与感”“共创感”“归属感”叙事
- 不要只写促销
- 尽量体现用户与品牌共同完成内容或体验
